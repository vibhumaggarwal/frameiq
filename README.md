# FRAMEIQ

AI-powered video intelligence platform. Upload a video and get:
- 🖼️ Frame extraction at configurable intervals
- 🤖 AI scene descriptions via Claude
- 🔍 Semantic search across all frames
- 💬 Natural language Q&A about your video content

## Setup

Set the `ANTHROPIC_API_KEY` secret in the Space settings to enable AI descriptions and search.
