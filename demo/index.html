Features

Everything you need

Built to be lean, accurate, and cost-efficient.

⚡
Aggressive caching
SHA256 video fingerprint ties results to files. Process a video twice and pay $0 — every description is reused from disk.
cache/<hash>_descriptions.json
🎯
Dedup-aware extraction
Consecutive frames compared via mean absolute difference. Near-identical frames are silently skipped — no wasted API calls.
opencv MAD threshold
🔒
Zero re-encoding
ffmpeg muxes the SRT subtitle track using -c copy. Your video quality is mathematically identical to the original.
output .mkv
🔍
Local semantic search
Sentence-transformers all-MiniLM-L6-v2 runs 100% locally. FAISS cosine-similarity index over every scene. No cloud required.
FAISS IndexFlatIP
🤖
Streaming RAG
Ask any question about the video. Claude receives all scene descriptions as context and streams a natural-language answer with timestamp citations.
claude-haiku-3-5
💰
Transparent costs
FRAMEIQ calculates your exact API cost before making any calls. Uses claude-haiku — the cheapest vision model at ~$0.0004/frame.

Get Started

Up in 60 seconds

Install dependencies, set your API key, and run.

📦 requirements.txt
# Install all deps
pip install -r requirements.txt

# You also need ffmpeg
brew install ffmpeg

# Set API key
export ANTHROPIC_API_KEY=sk-ant-...
🚀 Usage
# Full pipeline
python frameiq.py process movie.mp4

# Semantic search
python frameiq.py search movie.mp4 "..."

# Ask a question (RAG)
python frameiq.py ask movie.mp4 "..."

# Re-embed subtitles only
python frameiq.py embed movie.mp4

Built with ♥ using Claude claude-haiku · ffmpeg · FAISS · sentence-transformers
